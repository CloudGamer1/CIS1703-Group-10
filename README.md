# CIS1703-Group-10
CIS1703 group project group 10- This is a very simple to use GUI based "SmartStock" system. It allows for managing stock, including adding, removing and modifing aspects of the products. 

To use it you simply need to choose what type of product you want to add such as "Product" - normal product, "Perishable" - ones with expiery data, as well as "Electronic" - electronic products. Then feel out the fields needed (those withouth brackets() mean are mandatory, brackets explain for which type they are required.)

There are then buttons such as add products, remove it, cahnge quantity (requires quantatiy field to be filled out), smart alerts (produces list of products on low stock, as well as products close to expiery date)

There are also keybind capabilities "ENTER" allows to add product, and "CTRL" + "S" alows for saving the file.
